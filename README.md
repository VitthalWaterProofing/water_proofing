# WaterProofing Agency Backend

This is the backend for the WaterProofing Agency, built with Node.js, Express, TypeScript, and MongoDB.

## Getting Started

1.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up your `.env` file (copied from the original or populated with your MongoDB connection string).
4.  Run the development server:
    ```bash
    npm run dev
    ```

## Initial Setup
To set up the initial super admin account, run:
```bash
npm run setup:admin
```
This will create an admin user based on the seeder script.
